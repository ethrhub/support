## What is EthrHub?

EthrHub is a cloud-based network performance monitoring platform that helps you measure and analyze network performance across your infrastructure. Monitor bandwidth, latency, packet loss, and connection metrics in real-time from a centralized dashboard.

**Key Benefits:**
- **Easy to Deploy** - No complex setup, just run the agent
- **Centralized Monitoring** - View all your network tests in one place
- **Historical Analysis** - Track performance trends over time
- **Secure** - OAuth authentication and encrypted communication
- **Scalable** - From a few agents to hundreds across your infrastructure
